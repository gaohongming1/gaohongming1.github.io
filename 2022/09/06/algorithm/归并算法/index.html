# 归并算法

# 算法描述

归并排序是建立在归并操作上的一种有效的排序算法。该算法是采用分治法（Divide and Conquer）的一个非常典型的应用。

将已有序的子序列合并，得到完全有序的序列；即先使每个子序列有序，再使子序列段间有序。若将两个有序表合并成一个有序表，称为 2 - 路归并。

​ 上面是引用百度百科的解释。

​ 我觉得归并算法的主要意义在于合并这一过程，与二分算法、分支算法很相似，三者不分你我。

# 通用模型伪代码

  1. 对入参进行处理，构造临时数组 (用来合并过程中排序对比使用)
  2. 进入分割函数
    1. 判断边界值 left == right
    2. 找到中位值进行分割
    3. left 和 right 区间分别进入分割函数进行递归
    4. left 和 right 递归结束代表左右两个子区间求解完毕
    5. 进行左右两个子区间合并求解 ->3
  3. 进入合并求解函数
    1. 这里合并求解就是具体对两个区间进行的操作具体是根据题意写求解过程就行

# 区间合并判断

​ 这一类型的主要都是给定一个数组，给定一个判断条件，然后求解数组上满足这个条件的组合，记住这个组合一定有三种特性才能套用这套逻辑

  1. 满足给定特性的值肯定是左右两个区间配合求解的
  2. 满足给定特性的值肯定是有递增或者递减的一种组合规律的
  3. 对两个区间进行合并后需要对 [left,right] 整个区间进行排序 或者是在求解的过程中就进行排序
